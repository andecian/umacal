# UMA Monad Calculator

A web-based simulator to visualize arithmetic operations based on "UMA Monad Theory."

## Overview
This tool visualizes how division by zero and infinity behave as geometric mappings rather than errors.

- **Division by Zero**: Mapped to an orthogonal "Portal Dimension" (y(i)-axis).
- **Infinity Support**: Handles calculations like $\infty - \infty = 0$ based on the theory's rules.
- **Z-axis Visualization**: Displays the "existence layer" (depth of information) as the height of the bars.

## How to Use
1. Download `index.html`.
2. Open the file in any modern web browser.
3. Enter numbers or `inf` and select an operator to see the result and visualization.

## Background
Based on the "UMA Monad Theory" proposed by Hiroshi Miyauchi (2026), focusing on the philosophical and mathematical redefinition of zero and infinity.
